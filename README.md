# Sales Predictor 📈

A simple machine learning project that predicts future sales using historical sales data.

This project uses Linear Regression from Scikit-learn to forecast upcoming sales trends and visualize the results using Matplotlib.

---

# Features

- Load sales data from CSV
- Train a Linear Regression model
- Predict future sales
- Visualize sales trends
- Beginner-friendly project

---

# Technologies Used

- Python
- Pandas
- Matplotlib
- Scikit-learn
